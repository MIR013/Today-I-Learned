 * 주제는 프로그래밍에 초점을 두지만 한정하지 않는다.
 > 주제의 예시

  > 2. 수업에서 배운 것
  > 3. 철학
  > 4. 역사
  > 5. 기타
# Rule

 * 오늘 배운 것을 syntax: [timeSequence]()폴더에 .md파일로 3줄 요약한다.
 * 오늘 배운 것을 주제에 따라 폴더를 나누어 저장한다.
 * 오늘 배운 것에 코드가 있을 경우 첨부한다.
