# 2. 띄어쓰기와 엔터
#### 띄어쓰기
 * 띄어쓰기를 여러번 한 것은 한번 한 것과 같다<br>
 >띄 어 쓰 기 1 번<br>
  띄  어  쓰  기  2  번<br>
  띄   어   쓰   기   3   번<br>
 
#### 엔터
 * 엔터 1번: 글이 문단으로 붙는다
 * 엔터 2번: 글이 다음 단락으로 바뀐다.
 * 단, 엔터를 여러번 친다고 여러줄이 바뀌는 것은 아니다.

***
   첫머리 글
   엔터 1번으로 글이 문단으로 붙는다
   
   엔터 2번으로 글이 다음 단락으로 넘어간다
   
   
   
   엔터 4번으로 여러줄이 바뀌지는 않는다
***
 * 강제로 여러번 띄우고 싶을 경우 <br/>을 사용한다.
