# Ansible
---------------
자바를 기반으로한 GoF 디자인 패턴에 대해 공부한다.

--------------------------
#### 디자인 패턴 사용 이유
 - 효율적인 프로그래밍
 - 적절한 클래스는 하나의 관심사항에 대해 소수의 클래스 내에서 집중할 수 있게 한다. 
 - 다양한 추가 요구사항에 대해 더 쉽게 코드를 수정할 있게하기 위함
 
#### 유의할 점
 - 필요한 만큼 사용해야 한다.
 - 적용 후 코드가 수정하기 힘들어 졌다면 잘못될 것
